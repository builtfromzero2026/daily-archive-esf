اگر بخوایی تعداد اون متغیر قابل پیشمایشت رو در بیاری باید از فیلتر لنث استفاده کنی

like this: {{ name|length }}

if you want to change your character to uppcase you should use this filter: upper

like this: {{ name|upper }}

if you want to change first character to uppcase you should use this filter: title

like this: {{ name|title }}

if the value is none or nothing, this default value show.

like this: {{ name|default:"هرچی اینجا بگذاریم نمایش داده میشه ولی بشرطی که خالییییییی باشههههه"}}

like this: {{ name|yesno:"این متغیر خالی نیست و اوکیه , این متغیر خالیه لطفا تکمیلش کنید" }}
